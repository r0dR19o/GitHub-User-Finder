# GitHub User Finder

**Автор:** Родион Нагорнов

### Описание
Десктопное приложение для быстрого поиска пользователей GitHub. Позволяет просматривать основную информацию и сохранять интересных разработчиков в локальный список избранного.

### Технологии
- **Язык:** Python 3.x
- **GUI:** Tkinter / CustomTkinter
- **API:** GitHub Rest API
- **Хранение данных:** JSON

### Как использовать API
Приложение использует публичный эндпоинт: `https://github.com{username}`. 
*Примечание: Лимит анонимных запросов составляет 60 в час.*

### Инструкция по запуску
1. Склонируйте репозиторий: `git clone https://github.com`
2. Установите зависимости: `pip install requests`
3. Запустите: `python main.py`

### Примеры использования
1. Введите логин (например, `vinta`) в текстовое поле.
2. Нажмите кнопку "Поиск".
3. Если пользователь найден, нажмите "Добавить в избранное", чтобы сохранить его в файл `favorites.json`.
